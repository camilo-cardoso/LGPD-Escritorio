Este projeto é um sistema simples e eficiente criado com **Python + Streamlit**, voltado para ajudar **microempresários, pequenas empresas, escolas e ONGs** a lidar com documentos PDF de forma prática, segura e alinhada com boas práticas de **cibersegurança e LGPD**.

A aplicação oferece três funcionalidades principais:

---

## �� Funcionalidades

### 1️⃣ Unificar PDF  
Você seleciona vários arquivos PDF e o sistema gera um PDF único automaticamente.  
Ideal para montar dossiês, contratos e relatórios consolidados.

### 2️⃣ Extrair Texto do PDF  
Extrai todo o conteúdo textual de um arquivo PDF.  
Perfeito para análises, auditorias, organização de dados e busca de informações.

### 3️⃣ Comprimir PDF  
Reduz o tamanho do arquivo PDF mantendo boa qualidade.  
Ótimo para enviar documentos por e-mail ou sistemas com limite de upload.
